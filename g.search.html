DESCRIPTION

r.search searches for given keyword in GRASS modules name, description and keywords.

NOTES

There can be more keywords, g.search will search for each of them

EXAMPLE

Search all modules, where keywords buffer OR clip can be found 
g.search buffer,clip
Search all modules, where keywords overlay AND clip can be found 
g.search -a overaly,clip
